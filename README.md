# UDP-Multicast

Simple UDP multicast with C++ or C

## Pre-requisites

LZ4  
cmake > 3.0

## Compile

Compile LMDB

```
cd dependencies/lmdb/libraries/liblmdb/
make
```

Compile project

```
cmake -B .
make
```
